A simple and flexible bill-splitting application that helps you fairly divide bills between people, even when everyone pays differently.

Perfect for group meals, trips, or any shared expense.

---

## Features

- **Add Bills**
  - Add one or multiple bill items
  - Each item contributes to the total amount

- **Discounts**
  - Apply discounts to reduce the total bill
  - Supports fixed discounts

- **Fees / Taxes / Tips**
  - Add extra charges such as service fees, tax, or tips
  - Automatically included in the final calculation

- **People & Advanced Split Types**
  - Add multiple people to the bill
  - Each person can use a different split method

- **Export Result**
  - Export the final result as a **PNG image**
  - Easy to share via chat apps or social media

---

## How It Works

1. Add bill items
2. Apply discounts and extra fees (tax, tip, service charge)
3. Add people and choose a split type for each person
4. The app calculates the final amount per person
5. Export the result as a PNG for sharing
